## ⚠️ 重要提醒

1. **备份原文件**：修改前务必备份原始PVF文件
2. **格式规范**：严格遵循PVF文件格式规范，字符串使用反引号(\`)包围
3. **测试验证**：每次修改后都要进行游戏内测试
4. **版本兼容**：注意不同版本间的技能文件差异
