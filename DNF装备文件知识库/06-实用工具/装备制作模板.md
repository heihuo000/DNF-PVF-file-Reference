# DNF装备文件制作模板

## 📖 概述

本文档提供了各种类型装备的制作模板，帮助快速创建标准化的装备文件。每个模板都包含了必需的标签和常用的可选标签，可以根据需要进行修改。

---

## ⚔️ 武器类模板

### 基础武器模板
```
// ==================== 基本信息 ====================
[name] 武器名称
[name2] English Name
[basic explain] 基本说明
[detail explain] 详细说明
[flavor text] 风味文本

[grade] 装备等级(0-5)
[rarity] 稀有度(0-4)
[spectrum] 残影效果(0-1)

[attach type] [trade]  // [free]/[trade]/[sealing]/[trade delete]
[usable job] [职业代码]  // [swordman]/[fighter]/[gunner]/[mage]/[priest]/[thief]/[knight]/[demonic swordman]/[all]
[minimum level] 最低等级
[equipment type] [weapon]
[sub type] [武器子类型]  // [sword]/[katana]/[zanbato]/[lightsaber]/[staff]/[rod]/[pole]/[spear]/[bow]/[bowgun]/[auto gun]/[hand cannon]/[revolver]/[musket]/[claw]/[tonfa]/[boxing]/[gauntlet]/[cross]/[rosary]/[totem]/[scythe]/[wand]

// ==================== 装备属性 ====================
[equipment physical attack] +物理攻击力
[equipment magical attack] +魔法攻击力

[physical attack] 力量增加
[magical attack] 智力增加
[physical defense] 体力增加
[magical defense] 精神力增加

[HP MAX] HP最大值增加
[MP MAX] MP最大值增加

[critical hit] 物理暴击率
[magical critical hit] 魔法暴击率
[accuracy] 命中率
[attack speed] 攻击速度
[cast speed] 施放速度

// ==================== 属性强化 ====================
[fire attack] 火属性强化
[water attack] 水属性强化
[dark attack] 暗属性强化
[light attack] 光属性强化

// ==================== 特殊效果 ====================
[skill levelup] 技能ID 等级提升
[skill data up] 技能ID 数据索引 修改值

[critical damage] 暴击伤害增加
[human damage rate] 对人类伤害增加
[fort damage rate] 对要塞伤害增加
[monster damage rate] 对怪物伤害增加

[HP drain] HP吸收百分比
[MP drain] MP吸收百分比

[attack property] [属性类型]  // [fire]/[water]/[dark]/[light]
[push power] 推力
[lift power] 浮空力

// ==================== 状态异常 ====================
[active status control info]
    [bleeding] 概率 等级 持续时间
    [freeze] 概率 等级 持续时间
    [stun] 概率 等级 持续时间
    [poison] 概率 等级 持续时间

// ==================== 套装效果 ====================
[set name] 套装名称
[set item] 物品ID1 物品ID2 物品ID3 物品ID4 物品ID5
[set ability]
2 [效果标签] 数值
3 [效果标签] 数值
5 [效果标签] 数值

// ==================== 视觉音效 ====================
[icon] equipment/weapon/图标文件.img
[move wav] sound/weapon/移动音效.wav
[use wav] sound/weapon/使用音效.wav

[animation job] [职业代码]
[variation] 动画变化编号

// ==================== 其他属性 ====================
[weight] 重量(克)
[durability] 耐久度
[price] 价格
[cool time] 冷却时间(毫秒)

[limit upgradable level] 最大强化等级
[anti evil] 驱魔数值

[packagable] true/false
[exchangeable] true/false
```

## 📝 使用说明

### 1. 模板使用步骤
1. 选择合适的装备类型模板
2. 复制模板内容到新文件
3. 修改装备名称和基本信息
4. 设置装备属性和特殊效果
5. 配置视觉音效资源
6. 保存为 `.equ` 文件

### 2. 自定义修改
- 删除不需要的标签行
- 根据装备特色添加特殊效果
- 调整数值以保持游戏平衡
- 确保文件路径正确

### 3. 验证检查
- 检查标签格式是否正确
- 确认数值范围合理
- 验证文件编码为UTF-8
- 测试装备在游戏中的效果
