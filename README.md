## ⚡ 重要格式要求

> ⚠️ **关键提醒**：所有PVF文件必须遵循以下格式规范

- **参数分隔符**：使用 `Tab` 制表符，不是空格
- **缩进方式**：使用 `Tab` 制表符进行缩进
- **行结束符**：使用 `CRLF` (Windows格式)
- **字符串格式**：使用反引号 `` `字符串内容` ``
- **数值格式**：直接写数值，不加引号
